X EQUALIZER
=========

Chrome extension and Firefox addon for Graphic Equalizer.  
This uses Web Audio API Library ([XSound](https://github.com/Korilakkuma/XSound)).

## 1. Get repository

```bash
$ git clone git@github.com:Korilakkuma/X-EQUALIZER.git
$ cd X-EQUALIZER
$ npm install
$ npm run build
```
